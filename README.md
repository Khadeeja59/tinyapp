# TinyApp Project

TinyApp is a full stack web application built with Node and Express that allows users to shorten long URLs (Ã  la bit.ly). A registered user can create short URLS for the long URLS they have entered. They can also delete and update the URLS entered by them.

## Dependencies
- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.
